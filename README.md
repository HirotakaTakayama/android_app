ここでは、とある課題で作成した、アイドルと一緒に写真を撮れるアプリのコードを公開している。実際のところ、マニフェストとxmlファイルが入っていない為、これだけでは動作しないが、そこは殆ど追記せずに出来る為、必要であれば各自で作って欲しい。各コードの紹介を簡単に行う。・CameraPreviewクラスCameraPreviewクラスでは、SurfaceViewクラスでカメライメージの取得を行っている。・OverLayViewクラスOverLayViewクラスでは、用意した画像を読み取り、ViewクラスでonDrawメソッドをオーバーライドして画面上に画像を表示している。・MainActivityMainActivityでは、上記2つのクラスをインスタンス化し、カメライメージの上に橋本環奈の画像をオーバーレイしている。タッチイベントはここで実装されており、タッチした時に、オートフォーカスの実行とプログレスダイアログの表示をし、オートフォーカス完了のコールバックが来た後に保存の為に空のキャンバスを生成し、そこへカメライメージと橋本環奈画像を載せ、保存処理を行う。保存処理が完了し、Previewを再開した時にプログレスダイアログを閉じる。
